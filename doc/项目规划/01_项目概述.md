# Aiden-Lite 智能管家项目概述

## 项目目标
构建一个本地化部署的 AI 驱动智能管家系统，具备自然语言对话、场景识别、设备联动等能力，可与云端大模型无缝集成，支持多端扩展。

## 核心功能需求

### 1. AI 对话核心 (★★★★★)
- **语音交互**：语音输入 → 文字 → 理解 → 生成 → 文字 → 语音输出
- **自然对话**：支持连续多轮对话，带有上下文记忆
- **个性化交互**：识别访客/主人身份后，根据关系信息个性化应答
- **情感传达**：语音语调自然流畅，符合社交场景

### 2. 人脸识别与身份识别 (★★★★☆)
- 实时摄像头捕获
- 人脸检测与识别
- 从数据库检索身份信息（称谓、关系、职业等）
- 基于身份信息的个性化问候

### 3. 日程管理 (★★★★☆)
- 语音创建日程：自然语言理解日程信息
- 日程存储：本地数据库
- 自动提醒：基于摄像头识别时间、人脸识别触发提醒
- 日程分析：AI 智能拆分和规划

### 4. 唤醒与持续对话 (★★★★☆)
- 唤醒词识别（本地轻量级检测）
- 持续对话模式
- 对话超时自动待机

### 5. 智能设备联动 (★★★★☆)
- **MQTT 协议支持**：控制智能家居设备
- **米家生态集成**：调用小米设备、获取房间/设备信息
- **指令执行引擎**：预定义或动态生成的控制指令
- **场景自动化**：基于识别场景自动执行设备操作

### 6. 第三方工具集成 (★★★☆☆)
- **小爱同学集成**：需求分析后调用小爱执行特定指令
- **摄像头访问**：调用本地摄像头或 IP 摄像头
- **MCP 接口**：标准化协议支持未来扩展

### 7. 数据库与记忆系统 (★★★★☆)
- 用户信息库
- 日程库
- 对话历史记录
- 设备配置库
- 场景配置库

## 项目规模评估
- **阶段一（MVP）**：核心对话、人脸识别、基础日程 → 3-4 个月
- **阶段二**：设备联动、米家集成 → 2-3 个月
- **阶段三**：高级功能、优化体验 → 2 个月

## 非功能性需求
- **性能**：语音识别延迟 < 2s，人脸识别 < 1s
- **可靠性**：24/7 运行，网络掉线自动降级
- **扩展性**：模块化设计，支持新增功能
- **隐私安全**：本地数据加密、网络通信加密
